## ğŸ® How To Play (Cara Memainkan)

Di **Nimons Cooked**, kamu akan menjalankan peran sebagai chef yang harus mengelola dapur dan menyelesaikan pesanan pelanggan sebelum waktu habis. Berikut panduan dasar untuk memainkan game ini:

### Kontrol Gerak dan Interaksi

| Tombol | Fungsi |
|--------|--------|
| **W / A / S / D** | Bergerak ke atas, kiri, bawah, dan kanan |
| **SHIFT** | Dash/Sprint (bergerak lebih cepat) |
| **SPACEBAR** | Melakukan aksi/interaksi dengan station |
| **F** | Switch antara Chef 1 dan Chef 2 |
| **ESC** | Pause game |

### Alur Permainan

1. **Main Menu**: Pilih stage Sushi untuk memulai permainan
2. **Gameplay Flow**:
   - Lihat pesanan (order) yang masuk di bagian kiri layar
   - Kumpulkan bahan dari **Ingredient Storage Station**
   - Potong bahan di **Cutting Station** (sesuai resep)
   - Masak bahan di **Cooking Station** dengan utensil yang tepat (Frying Pan atau Boiling Pot)
   - Rakit hidangan di **Assembly Station/Ingredient Storage Station**
   - Sajikan pesanan ke **Serving Station** untuk menyelesaikan order
   - Cuci plate di **Washing Station**
3. **Time Management**: Selesaikan pesanan sebelum timer habis untuk mendapat poin
4. **Scoring**: Kumpulkan poin dari setiap pesanan yang berhasil diselesaikan
5. **Result Screen**: Lihat hasil akhir permainan (PASS/FAIL) berdasarkan skor total
